# ViaoScrap

Première version du projet **ViaoScrap**.

## Objectif

Ce dépôt rassemble la base du projet afin de publier une première version stable et documentée.

## Contenu de la première version

- Documentation initiale du projet.
- Journal des versions pour suivre les évolutions.
- Script de démarrage minimal pour valider la première version.

## Démarrage rapide

1. Cloner le dépôt.
2. Lancer le script de démarrage :

```bash
python src/main.py
```

3. Consulter la documentation ci-dessus pour comprendre le périmètre de la version initiale.
